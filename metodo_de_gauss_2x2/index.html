Reglas de uso

  • Numeros negativos:

    Para poner un numero negativo, se debe colocar el signo - adelante de dicho numero. Ejemplo: 4 negativo se espresaría -4

  • Fracciones:

    Para expresar fracciones debe colocarse el signo / para señalar la division. Ejemplo: 3 cuartos se expresaría 3/4. En el caso de poner un numero con coma, debe ponerse el numero en cuestion con un punto en vez de una coma
